Constructor

A Constructor is a special type of function or method.The main function of Constructor is to initialize all the variables of class, but it can performed other piece of code that we want to get executed. Constructor has some of the special features which are discussed below :
1.It has no return type
2.It has same name as that of the classname
3.Constructor can be overload. 4.It is executed when object of that class is created

Types of constructors are discussed below
1.Default Constructor
2.Non-Parameterized Constructor
3.Parameterized Constructor
4.Overloaded Constructor

Default Constructor

This Constructor get automatically executed when the program is get compliled.

Non-Parameterized Constructor

It does not have any parameter and it can executed when objects are created.
class Main(){
Main(){
System.out.println("This is Constructor");
}
public static void main(String[] args){
Main m = new Main();
}
}

Parameterized Constructor

It contains some parameters in its structure.
class Main(){
Main(int a,String s){
System.out.println(a + "and" + s);
}
public static void main(String[] args){
Main m = new Main(23,"java");
}
}

Overloaded Constructor

The name of Constructor is same but it defers in the number,type and format of the parameter list.
class Main(){
Main(){
System.out.println("This is constructor");
}
Main(double d){
System.out.println(d);
}
Main(int a,String s){
System.out.println(a + "and" + s);
}
public static void main(String[] args){
Main m = new Main(23,"java");
}
}
