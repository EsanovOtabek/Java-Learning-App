Interface

An interface is like a class but not a class.It has only static constants and abstract methods.A class implements a interface.It acheives the Abstraction and supports multiple inheritence in java.Syntax is :
interface name{}

interface I{
void m1(); //It is by default abstract
}
class A implements I{
public void m1(){
System.out.println("Hiii m1");
}
public static void main(String[] args){
A obj = new A();
obj.m1();
}
}
Output:
Hiii m1

Packages

A package in java is a group of classes,interfaces and sub-packages.There are two types of packages which are user-defined packages and built-in packages.Packages can be use to store interfaces and classes within it.It provides access protection also.'package' keyword is used to create a package in java

package mypackage;
public class A{
public static void main(String[] args){
System.out.println("Hello Package");
}
}
Output:
Hello Package

'import' keyword is used to access the classes inside a package
1.import mypackage.*;
this will import all the classes inside mypackage
2.import mypackage.ClassName;
this will import only class ClassName


Access Modifiers

The access modifiers in java specifies the accessibility or scope of variable,method,constructor or class. There are 4 types of access modifiers in java:
1.public
2.protected
3.private
4.default

1.The public access modifier is accessible anywhere, it has very wide scope.It gives accessibility outside packages, classes etc.

2.The protected access modifier is accessible within package and outside package by inheritence.

3.The private access modifier gives accessibility only within a class.

4.If any access modifier is not used then default access modifier is there bydefault and it is accessible only within the package.
