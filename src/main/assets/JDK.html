Java Development Kit (JDK)

The JDK is the software for developing and running Java programs. The JDK consists of a set of separate programs, each invoked from a command line, for developing and testing Java programs. Java Runtime Environment (JRE)

JRE stands for the Java Runtime Environment.It provides all the requirements for executing a particular Java Application.JRE consists of Java Virtual Machine,classes and important libraries and files.

Java Virtual Machine (JVM)

Java Virtual machine is mothing but a software program that runs and convert java program or code into a optimized set of instructions known as bytecode.The main method of java program is also invoked by the java virtual machine.
