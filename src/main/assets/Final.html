static

'static' means it creates only one single copy.It can be applied with variables,methods,blocks and classes.
It can be used to refer to common properties of all objects,static variable gets memory allocated only once at the time of class loading.It makes program's memory efficient.
A static method is belongs to the class and not to the object of that class.It can be call with the help of class by using dot operator.Static method can access static data.

final

final keyword provides restrictions.It can be applied to variable,method and class.
If you make a final variable then you cannot change the value of that variable.It will be a constant variable.
If you make a final method then you cannot override that method.
If you make a final class then you cannot extend that class.

this

'this' keyword refers to the current object.
It can be use to refer current object
It can be use to call current class variable and method
this() is use to call current class constructor
'this' keyword is used to pass as argument in method and constructor
It is used to return object of current class.
