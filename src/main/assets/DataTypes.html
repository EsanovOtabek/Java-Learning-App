Primitive Data Types

byte
- Min value : -128
- Max value : 127
- Default value : 0
It is a 8-bit signed integer

short
- Min value : -32,768
- Max value : 32,767
- Default value : 0
It is a 16-bit signed integer

int
- Min value : -2,147,483,648
- Max value : 2,147,483,647
- Default value : 0
It is a 32-bit signed integer

long
- Min value : -9,223,372,036,854,775,808
- Max value : 9,223,372,036,854,775,807
- Default value : 0
It is a 64-bit signed integer

char
- Min value : '\u0000'(0)
- Max value : '\uffff'(65,535)
It is a 16-bit unicode character

float
The float data type is a single-precision 32-bit IEEE 754 floating point

double
The double data type is a double-precision 64-bit IEEE 754 floating point

boolean
boolean data type represents only one bit of information either true or false
