Inheritence in Java

Inheritence is very important concept in OOPs.By inheritence you can built new classes which holds the properties of other class(i.e its variables and methods). The new class is known as sub-class while the class whose properties it hold is known as super class.Inheritence is use for code reusability and method overriding.
The syntax of Inheritence in java is :
class Subclass-name extends Superclass-name{
//methods and variables
}
Types of Inheritance in java are:
1.Single
2.Multilevel
3.Heirarical
4.Multiple

Multiple Inheritance is not supported in java

Single Inheritence
class A{
void m1(){
System.out.println("I am m1");
}
}
class B extends A{
void m2(){
System.out.println("I am m2");
}
}
class Main{
public static void main(String[] args){
B b = new B();
b.m1();
b.m2();
}
}
Output:
I am m1
I am m2

Multilevel Inheritence
class A{
void m1(){
System.out.println("I am m1");
}
}
class B extends A{
void m2(){
System.out.println("I am m2");
}
}
class C extends B{
void m3(){
System.out.println("I am m3");
}
}
class Main{
public static void main(String[] args){
C c = new C();
c.m1();
c.m2();
c.m3();
}
}
Output:
I am m1
I am m2
I am m3

Heirarical Inheritence
class A{
void m1(){
System.out.println("I am m1");
}
}
class B extends A{
void m2(){
System.out.println("I am m2");
}
}
class C extends A{
void m3(){
System.out.println("I am m3");
}
}
class Main{
public static void main(String[] args){
C c = new C();
c.m1();
// c.m2() it will give an error.
c.m3();
}
}
Output:
I am m1
I am m2
