1. if

An if statement executes the statements if the condition is true. A one-way if statement executes an action if and only if the condition is true. The syntax for a one-way if statement is:
if(boolean-expression){
statement(s);
}
If the boolean-expression evaluates to true, the statements in the block are executed, otherwise the control of program goes to next statements.

2. if-else

An if-else statement decides which statements to execute based on whether the condition is true or false. A one-way if statement takes an action if the specified condition is true. If the condition is false, nothing is done. But what if you want to take alternative actions when the condition is false then you can use a two-way if-else statement.
Here is the syntax for a two-way if-else statement:
if (boolean-expression) {
statement(s)-for-the-true-case;
}
else {
statement(s)-for-the-false-case;
}

3. Nested if and multi-way if-else

An if statement can be inside another if statement to form a nested if statement. The inner if statement is said to be nested inside the outer if statement. The inner if statement can contain another if statement; in fact, there is no limit to the depth of the nesting. For example, the following is a nested if statement:
if (i > k) {
if (j > k)
System.out.println("A");
}
else
System.out.println("B");

The nested if statement can be used to implement multiple choices.
if (score >= 90.0)
grade = 'A';
else if (score >= 80.0)
grade = 'B';
else if (score >= 70.0)
grade = 'C';
else if (score >= 60.0)
grade = 'D';
else
grade = 'F';
