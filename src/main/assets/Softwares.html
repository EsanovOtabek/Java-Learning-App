Softwares of Java

1.JDK(Java Development Kit)

For running the java program the JDK is neccessary.It includes all packages,classes interfaces,libraries,compiler,JVM and JRE used to write,compile and execute the java programs. All other Integrated Development Environment needs JDK for running java program.

2.NetBeans

NetBeans is Integrated Development Environment(IDE) for Java.NetBeans allows user to run java programs,java applets as well as desktop applications.It was devloped by Apache Software Foundation and Oracle Corporation.It has other extensions also like C,C++,HTML5 and Javascript.

3.Eclipse

Eclipse is also an Integrated Development Environment(IDE) which is mostly used for java.It has a good plugin system it also allows to make desktop as well as android applications.It was developed by Eclipse Foundation.

4.Intellij Idea

Intellij is an java Integrated Development Environment for making Softwares and desktop applications. It has two editions namely community edition and commercial edition.It holds language like java as well as kotlin.The Software "Android Studio" is made on Intellij.It was developed by JetBrains.
