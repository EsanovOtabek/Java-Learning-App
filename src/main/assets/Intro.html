Java was developed by Sun Microsystems in 1991 and now it is acquired by Oracle Corporation.Java was developed by Games Gosling and Patrick Naughton.
The target of Java is to write a program once and then run this program on multiple operating systems. Java is Object Oriented language and holds properties of Object Oriented Programming like

1.Encapsulation
2.Abstraction
3.Inheritence
4.Polymorphism

Here are some of the features of Java :
Simple
The Java language is easy to learn. Java code is easy to read and write and run on any operating systems.

Object Oriented
Java has all properties of Object Oriented Technology such as Encapsulation,Abstraction,Inheritence and Polymorphism.

Robust and Secure
Java is very secure because of its security manager that allows checking at compile time and runtime, you have never heard that java application is attacked by virus.It has automatic garbage collection technology which makes it Robust and also it does not have the onloadedmetadata concept of pointers like C/C++

High Performance
Java code is firslty converted to bytecode by JVM which provides strong Performance.

Multithreaded
It means that we can create applications in java which are multitasking and highly interactive.

Platform Independent
Java code is converted to bytecode which can run on any operating systems like Windows,Solaris,Linux and MacOS.
