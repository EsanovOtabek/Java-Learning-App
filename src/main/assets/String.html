String

A string is an object which represents the sequence of characters.An array of characters is known as string. Java String class provides the lots of methods.String can be created as follows:
String s1 = "Java";
String s2 = "World";
String s3 = s1 + s2;// JavaWorld
String s = new String("India");

Methods of String class are:

1 char charAt(int index)
2 int length()
3 String substring(int beginIndex)
4 String substring(int beginIndex, int endIndex)
5 boolean contains(CharSequence s)
6 boolean equals(Object o)
7 boolean isEmpty()
8 String concat(String str)
9 String replace(char old, char new)
10 String replace(CharSequence old, CharSequence new)
11 static String equalsIgnoreCase(String s)
12 String[] split(String s)
13 String[] split(String s, int limit)
14 String intern()
15 int indexOf(int ch)
16 int indexOf(int ch, int fromIndex)
17 int indexOf(String substring)
18 int indexOf(String substring, int fromIndex)
19 String toLowerCase()
20 String toUpperCase()
21 String trim()
22 static String valueOf(int value)


StringBuilder

StringBuilder class is used to create immutable strings i.e the strings which can be modified.Syntax is:
StringBuilder s = new StringBuilder(s);
Methods of StringBuilder class are:

1 public StringBuilder append(String s)
2 public StringBuilder insert(int offset, String s)
3 public StringBuilder replace(int startIndex, int endIndex, String str)
4 public StringBuilder delete(int startIndex, int endIndex)
5 public StringBuilder reverse()
6 public int capacity()
7 public void ensureCapacity(int minimumCapacity)
8 public char charAt(int index)
9 public int length()
10 public String substring(int beginIndex)
11 public String substring(int beginIndex, int endIndex)
