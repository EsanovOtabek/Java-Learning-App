The java.io package contains all the classes required for performing file operations.This happens by stream.A stream is a sequence of data.We have output,input and error stream in java.
Java uses outputstream for writing the data in the file and inputstream for reading the content of the file. Stream is nothing but a connection.

OutputStream

outputstream class is an abstract class and had some following methods:
1. public void write(int) throws IOException - it writes the byte to current outputstream
2. public void write(byte[]) throws IOException - it writes the array of bytes to current outputstream
3. public void flush() throws IOException - flushes the output stream
4. public void close() throws IOException - closes the outputstream

InputStream

inputstream class is also an abstract class and had some following methods:
1. public abstract int read() throws IOException - reads the next byte from inputstream
2. public int available() throws IOException - returns the number of bytes available in stream
3. public void close() throws IOException - closes the input stream

Syntax for getting connection (stream) with a file is as follows:

FileOutputStream fos = new FileOutputStream("file_location");
FileInputStream fis = new FileInputStream("file_location");
