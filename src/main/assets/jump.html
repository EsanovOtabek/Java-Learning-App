1. break

It is used to terminate a loop.When the break statement is executed then control exits from current running loop (for,while,do-while) and goes to next statements after the loop.

2. continue

This keyword skips the body of loop and goes to the next iteration of loop.

3. return

This keyword is used to terminate the execution of a method by giving some value to the method depending upon its return type.

4. goto

This keyword is used to calling a label which holds specific code for execution whenever we need it.
